**Project Name: CarGuru Deal Rating Predictor**

**Description:**
This project aims to develop a machine learning model that predicts deal ratings for vehicles listed on the CarGuru website. The model utilizes data from the CarGuru API to train and make predictions on the likelihood of a particular listing being a good deal. By analyzing various features of the listings, such as vehicle make, model, year, mileage, and price, the model provides valuable insights to both buyers and sellers, helping them make informed decisions.

**Installation:**
1. Clone the repository from GitHub:
   ```
   git clone https://github.com/Hassa678/dealRating_car.git
   ```
2. Navigate to the project directory:
   ```
   cd CarGuru-Deal-Rating-Predictor
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
